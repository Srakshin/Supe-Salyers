<p> Clone the link </p>

```terminal
git clone https://github.com/Srakshin/Supe-Salyers
```
<p> Install Node.js </p>

```terminal
https://nodejs.org/en
```

```terminal
npm intall
```
<p>To Host the page and Edit Run npm </p>

```terminal
npm run start
```
